HTML Resources

Language Snapshot

Overview

  • Language Type: Markup Language
  • Key Features:
    • Structure Web Pages: Defines the structure of a webpage by organizing elements
    • Multimedia Embedding: Supports embedding images, videos, audio, and other multimedia elements
    • Cross-Browser Compatibility: Works across various web browsers
    • Responsive Design Support: Works with CSS and JavaScript for responsive designs
    • SEO-Friendly: Proper structure helps search engines index and understand content better

Invention

  • Creator: Tim Berners-Lee
  • Year Created: 1991
  • Current Version: HTML5 (released in 2014)
  • Major Influence: Creation and structure of web content
  • Purpose: To create and structure web content like text, images, and multimedia

Basic HTML Structure


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content
    ="width=device-width,
    initial-scale=1.0">
    <title>Page Title</title>
</head>
<body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
</body>
</html>

Popular HTML Elements

  • <h1> to <h6>: Define headings
  • <p>: Defines paragraphs
  • <a>: Creates hyperlinks
  • <img>: Embeds images
  • <div>: Defines a section or container

Common Use Cases

  • Building Websites: Every website is built with HTML
  • Creating Web Applications: Forms the foundation of web applications
  • Embedding Media: Allows embedding of images, videos, and audio on web pages
  • Structuring Content: Organizes web content for better readability and SEO

Cheatsheet

Basic Level

1. Basic HTML Structure

A simple structure to build any webpage with a title, heading, and paragraph.

            <!DOCTYPE html>
            <html>
            <head>
                <title>Page Title</title>
            </head>
            <body>
                <h1>This is a Heading</h1>
                <p>This is a paragraph.</p>
            </body>
            </html>

2. Headings

Defines different heading levels on the page. <h1> is the highest level.

            <h1>Main Heading</h1>
            <h2>Subheading</h2>

3. Paragraphs

Used to add blocks of text on the webpage.

            <p>This is a paragraph of text.</p>

4. Links

Creates a clickable link to another webpage.

            <a href="https://example.com">Click here</a>

5. Images

Embeds an image in your webpage.

            <img src="image.jpg" alt="Image description">

6. Ordered and Unordered Lists

Ordered lists are numbered, unordered lists are bullet points.

            <!-- Ordered (Numbered) List -->
            <ol>
                <li>First item</li>
                <li>Second item</li>
            </ol>
            
            <!-- Unordered (Bullet) List -->
            <ul>
                <li>Item 1</li>
                <li>Item 2</li>
            </ul>

Intermediate Level

1. Tables

Displays data in a structured format with rows and columns.

            <table>
                <tr>
                    <th>Header 1</th>
                    <th>Header 2</th>
                </tr>
                <tr>
                    <td>Data 1</td>
                    <td>Data 2</td>
                </tr>
            </table>

2. Forms

Used to collect user input and submit data.

            <form action="/submit" method="post">
                <label for="name">Name:</label>
                <input type="text" id="name" name="name">
                <input type="submit" value="Submit">
            </form>

3. Divisions and Spans

<div> groups block elements, and <span> groups inline elements.

            <!-- Div (Block-level) -->
            <div>This is a division.</div>
            
            <!-- Span (Inline-level) -->
            <span style="color:red;">Red Text</span>

4. Attributes

id and class attributes are used to uniquely identify and style elements.

            <p id="para1" class="text">This is a paragraph.</p>

5. Link Target Attribute

Opens links in a new browser tab.

            <a href="https://example.com" target="_blank">Open in new tab</a>

6. Comments

Comments are used for documentation within the code and are not displayed on the webpage.

            <!-- This is a comment -->

Expert Level

1. Video Embedding

Embeds a video file with control options like play, pause, and volume.

            <video width="320" height="240" controls>
                <source src="movie.mp4" type="video/mp4">
            </video>

2. Audio Embedding

Embeds an audio file with player controls.

            <audio controls>
                <source src="audio.mp3" type="audio/mpeg">
            </audio>

3. Semantic Elements

Semantic elements provide meaning to the structure of web content.

            <header>
                <h1>Website Title</h1>
            </header>
            <nav>
                <a href="#home">Home</a>
            </nav>
            <main>
                <p>Main content here.</p>
            </main>
            <footer>
                <p>Footer text here.</p>
            </footer>

4. Meta Tags

Used to provide metadata for the document, including character encoding and SEO information.

            <meta charset="UTF-8">
            <meta name="description" content="Description of the webpage.">
            <meta name="keywords" content="HTML, CSS, JavaScript">
            <meta name="author" content="John Doe">

Standard Code Snippets

Basic Level

Basic HTML Structure

A simple HTML5 document structure:

              <!DOCTYPE html>
              <html lang="en">
              <head>
                  <meta charset="UTF-8">
                  <meta name="viewport" content="width=device-width, initial-scale=1.0">
                  <title>Basic HTML Page</title>
              </head>
              <body>
                  <h1>Welcome to My Web Page</h1>
                  <p>This is a simple paragraph of text.</p>
              </body>
              </html>
Linking an External Stylesheet

How to link an external CSS file:

              <!DOCTYPE html>
              <html lang="en">
              <head>
                  <meta charset="UTF-8">
                  <meta name="viewport" content="width=device-width, initial-scale=1.0">
                  <link rel="stylesheet" href="styles.css">
                  <title>Stylesheet Example</title>
              </head>
              <body>
                  <h1>This page has an external stylesheet</h1>
              </body>
              </html>
Adding an Image

Inserting an image with alternative text and dimensions:

              <!DOCTYPE html>
              <html lang="en">
              <head>
                  <meta charset="UTF-8">
                  <meta name="viewport" content="width=device-width, initial-scale=1.0">
                  <title>Image Example</title>
              </head>
              <body>
                  <h1>My Favorite Image</h1>
                  <img src="image.jpg" alt="A description of the image" width="300">
              </body>
              </html>
Creating a Simple List

An unordered list of fruits:

              <h2>Fruits List</h2>
              <ul>
                  <li>Apple</li>
                  <li>Banana</li>
                  <li>Orange</li>
              </ul>
Hyperlinks with Target

Creating a link that opens in a new tab:

              <p>Visit <a href="https://www.example.com" target="_blank">this website</a> for more information.</p>

Intermediate Level

HTML Form with Input Fields

A simple form with text and email inputs:

              <form action="/submit-form" method="POST">
                  <label for="name">Name:</label>
                  <input type="text" id="name" name="name" required><br>
                  <label for="email">Email:</label>
                  <input type="email" id="email" name="email" required><br>
                  <input type="submit" value="Submit">
              </form>
Embedding a Video

Using the HTML5 video element:

              <video width="400" controls>
                  <source src="movie.mp4" type="video/mp4">
                  Your browser does not support the video tag.
              </video>
Creating a Table

A simple HTML table with headers and data:

              <table border="1">
                  <thead>
                      <tr>
                          <th>Name</th>
                          <th>Age</th>
                          <th>City</th>
                      </tr>
                  </thead>
                  <tbody>
                      <tr>
                          <td>John</td>
                          <td>28</td>
                          <td>New York</td>
                      </tr>
                      <tr>
                          <td>Jane</td>
                          <td>32</td>
                          <td>Los Angeles</td>
                      </tr>
                  </tbody>
              </table>
Creating a Drop-down List

A select element with options:

              <label for="cars">Choose a car:</label>
              <select id="cars" name="cars">
                  <option value="volvo">Volvo</option>
                  <option value="saab">Saab</option>
                  <option value="mercedes">Mercedes</option>
                  <option value="audi">Audi</option>
              </select>
Checkboxes and Radio Buttons

Examples of checkbox and radio button inputs:

              <h3>Select your favorite fruit:</h3>
              <label><input type="checkbox" name="fruit" value="apple"> Apple</label><br>
              <label><input type="checkbox" name="fruit" value="banana"> Banana</label><br>
              <label><input type="checkbox" name="fruit" value="orange"> Orange</label><br>
              
              <h3>Select your gender:</h3>
              <label><input type="radio" name="gender" value="male"> Male</label><br>
              <label><input type="radio" name="gender" value="female"> Female</label><br>

Expert Level

Creating a Responsive Navigation Bar

HTML and CSS for a simple responsive navigation:

              <nav>
                  <ul>
                      <li><a href="#home">Home</a></li>
                      <li><a href="#about">About</a></li>
                      <li><a href="#services">Services</a></li>
                      <li><a href="#contact">Contact</a></li>
                  </ul>
              </nav>
              
              <style>
                  nav ul {
                      list-style-type: none;
                      padding: 0;
                  }
              
                  nav ul li {
                      display: inline-block;
                      margin-right: 20px;
                  }
              
                  nav ul li a {
                      text-decoration: none;
                  }
              </style>
HTML5 Audio Player

Embedding an audio player:

              <audio controls>
                  <source src="song.mp3" type="audio/mp3">
                  Your browser does not support the audio element.
              </audio>
HTML5 Canvas Drawing

Creating a simple drawing on a canvas:

              <canvas id="myCanvas" width="400" height="300" style="border:1px solid #000000;"></canvas>
              
              <script>
                  var canvas = document.getElementById('myCanvas');
                  var context = canvas.getContext('2d');
                  context.fillStyle = 'blue';
                  context.fillRect(50, 50, 150, 100);
              </script>
Embedding Google Maps

Embedding a Google Maps iframe:

              <iframe
                  width="600"
                  height="450"
                  style="border:0"
                  loading="lazy"
                  allowfullscreen
                  src="https://www.google.com/maps/embed/v1/place?key=YOUR_API_KEY&q=Space+Needle,Seattle+WA">
              </iframe>
Creating a Responsive Image

Making an image responsive with CSS:

              <img src="image.jpg" alt="Description" style="max-width:100%;height:auto;">
