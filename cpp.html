C++ Resources

Language Snapshot

Overview

  • Language Type: General-purpose, multi-paradigm
  • Key Features:
    • Object-Oriented: Supports encapsulation, inheritance, and polymorphism
    • Generic Programming: Enables writing code that works with any data type
    • Standard Template Library (STL): Provides reusable components for common data structures and algorithms
    • Low-Level Memory Manipulation: Allows direct memory access and pointer arithmetic
    • High Performance: Optimized for system-level programming and resource-constrained environments

Invention

  • Creator: Bjarne Stroustrup
  • Year Created: 1979
  • Major Influence: Object-oriented programming, generic programming, systems programming
  • Influenced Languages: Java, C#, D

Notable Frameworks and Libraries

  • GUI: Qt, wxWidgets
  • Game Development: Unreal Engine, SFML
  • Networking: Boost.Asio, POCO
  • Machine Learning: TensorFlow, OpenCV

Common Use Cases

  • System Software: Operating systems, device drivers
  • Game Development: High-performance game engines and applications
  • Financial Systems: High-frequency trading platforms
  • Embedded Systems: IoT devices, real-time systems

Cheat Sheet

Basic Level

1. Basic Syntax

#include <iostream>: Include standard library for input/output.

using namespace std;: Avoid prefixing std:: before standard functions.

int main(): Entry point of the C++ program.

cout: Output stream to print text.

Example: cout << "Hello, World!";

cin: Input stream to get user input.

Example: cin >> variable;

2. Data Types

int, float, double, char, bool

Example: int a = 10;

3. Operators

Arithmetic Operators: +, -, *, /, %

Comparison Operators: ==, !=, <, >

Logical Operators: &&, ||, !

4. Control Flow

if-else: Conditional statements.

            if (condition) {
              // code
            } else {
              // code
            }

for, while, do-while: Loops.

            for (int i = 0; i < 10; i++) {
              cout << i;
            }

5. Functions

Defining and calling functions.

            int add(int a, int b) {
              return a + b;
            }

Intermediate Level

1. Arrays

Fixed-size collection of elements.

            int arr[5] = {1, 2, 3, 4, 5};

2. Pointers

Store memory address of variables.

            int a = 10;
            int *ptr = &a;

3. Structures

Custom data types.

            struct Student {
              string name;
              int age;
            };

4. File I/O

Read from and write to files.

            ofstream myFile("example.txt");
            myFile << "Hello!";

5. Object-Oriented Programming (OOP)

Classes and Objects: Blueprint of an object.

            class Car {
              public:
                string brand;
                int speed;
            };

6. Inheritance

One class inherits properties from another class.

            class Vehicle {
              public:
                int wheels;
            };
            
            class Car : public Vehicle {
              public:
                string brand;
            };

Expert Level

1. Dynamic Memory Allocation

Use of new and delete to allocate and free memory.

            int* ptr = new int;
            delete ptr;

2. Operator Overloading

Overload operators to work with custom objects.

            class Complex {
              public:
                int real, imag;
                Complex operator + (Complex const &obj) {
                  Complex res;
                  res.real = real + obj.real;
                  res.imag = imag + obj.imag;
                  return res;
                }
            };

3. Templates

Generalized functions and classes.

            template <typename T>
            T add(T a, T b) {
              return a + b;
            }

4. Exception Handling

Use try, catch, and throw to handle exceptions.

            try {
              throw 404;
            } catch (int e) {
              cout << "Error: " << e;
            }

5. Standard Template Library (STL)

Vectors: Dynamic arrays.

            vector<int> v = {1, 2, 3};
            v.push_back(4);

Maps: Key-value pairs.

            map<string, int> m;
            m["apple"] = 1;

6. Multithreading

Running multiple threads simultaneously.

            #include <thread>
            void function1() {
              cout << "Thread 1";
            }
            
            int main() {
              thread t1(function1);
              t1.join();
            }

Palindrome Check

Problem: Write a program to check if a given string is a palindrome.

        #include <iostream>
        using namespace std;
        
        bool isPalindrome(string str) {
            int start = 0;
            int end = str.length() - 1;
            while (start < end) {
                if (str[start] != str[end]) return false;
                start++;
                end--;
            }
            return true;
        }
        
        int main() {
            string str;
            cout << "Enter a string: ";
            cin >> str;
            cout << (isPalindrome(str) ? "Palindrome" : "Not a palindrome") << endl;
            return 0;
        }
Bubble Sort

Problem: Write a program to implement the Bubble Sort algorithm.

        #include <iostream>
        using namespace std;
        
        void bubbleSort(int arr[], int n) {
            for (int i = 0; i < n-1; i++)
                for (int j = 0; j < n-i-1; j++)
                    if (arr[j] > arr[j+1])
                        swap(arr[j], arr[j+1]);
        }
        
        int main() {
            int arr[] = {64, 34, 25, 12, 22};
            int n = sizeof(arr)/sizeof(arr[0]);
            bubbleSort(arr, n);
            cout << "Sorted array: ";
            for (int i = 0; i < n; i++)
                cout << arr[i] << " ";
            cout << endl;
            return 0;
        }

Expert Level

Merge Sort

Problem: Write a program to implement the Merge Sort algorithm.

        #include <iostream>
        using namespace std;
        
        void merge(int arr[], int left, int mid, int right) {
            int i, j, k;
            int n1 = mid - left + 1;
            int n2 = right - mid;
            int L[n1], R[n2];
        
            for (i = 0; i < n1; i++)
                L[i] = arr[left + i];
            for (j = 0; j < n2; j++)
                R[j] = arr[mid + 1 + j];
        
            i = 0;
            j = 0;
            k = left;
        
            while (i < n1 && j < n2) {
                if (L[i] <= R[j]) {
                    arr[k] = L[i];
                    i++;
                } else {
                    arr[k] = R[j];
                    j++;
                }
                k++;
            }
        
            while (i < n1) {
                arr[k] = L[i];
                i++;
                k++;
            }
        
            while (j < n2) {
                arr[k] = R[j];
                j++;
                k++;
            }
        }
        
        void mergeSort(int arr[], int left, int right) {
            if (left < right) {
                int mid = left + (right - left) / 2;
                mergeSort(arr, left, mid);
                mergeSort(arr, mid + 1, right);
                merge(arr, left, mid, right);
            }
        }
        
        int main() {
            int arr[] = {38, 27, 43, 3, 9, 82, 10};
            int n = sizeof(arr) / sizeof(arr[0]);
            mergeSort(arr, 0, n - 1);
            cout << "Sorted array: ";
            for (int i = 0; i < n; i++)
                cout << arr[i] << " ";
            cout << endl;
            return 0;
        }
Binary Search

Problem: Write a program to implement the Binary Search algorithm.

        #include <iostream>
        using namespace std;
        
        int binarySearch(int arr[], int size, int target) {
            int left = 0, right = size - 1;
            while (left <= right) {
                int mid = left + (right - left) / 2;
                if (arr[mid] == target)
                    return mid;
                if (arr[mid] < target)
                    left = mid + 1;
                else
                    right = mid - 1;
            }
            return -1;
        }
        
        int main() {
            int arr[] = {2, 3, 4, 10, 40};
            int n = sizeof(arr) / sizeof(arr[0]);
            int target = 10;
            int result = binarySearch(arr, n, target);
            if (result != -1)
                cout << "Element found at index: " << result << endl;
            else
                cout << "Element not found" << endl;
            return 0;
        }
Depth First Search (DFS) for a Graph

Problem: Write a program to implement the Depth First Search (DFS) algorithm on a graph.

        #include <iostream>
        #include <vector>
        using namespace std;
        
        void DFS(int node, vector<bool> &visited, const vector<vector<int>> &adj) {
            visited[node] = true;
            cout << node << " ";
        
            for (int neighbor : adj[node]) {
                if (!visited[neighbor]) {
                    DFS(neighbor, visited, adj);
                }
            }
        }
        
        int main() {
            int n = 5;
            vector<vector<int>> adj(n);
            adj[0] = {1, 2};
            adj[1] = {0, 3};
            adj[2] = {0, 4};
            adj[3] = {1};
            adj[4] = {2};
        
            vector<bool> visited(n, false);
            cout << "DFS traversal: ";
            DFS(0, visited, adj);
            cout << endl;
            return 0;
        }
