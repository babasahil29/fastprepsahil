Language Snapshot

Introduction to PHP

  • Language: PHP (Hypertext Preprocessor)
  • Invented By: Rasmus Lerdorf
  • Year: 1994
  • Overview: PHP is a widely-used open-source server-side scripting language designed for web development but also used as a general-purpose programming language. PHP is embedded in HTML, and it interacts with databases, making it a powerful tool for dynamic website development.

Key Features

  • Server-Side Language: Executes on the server, producing dynamic content on web pages.
  • Cross-Platform: Works on various platforms (Windows, Linux, macOS).
  • Database Interaction: Supports various databases like MySQL, PostgreSQL, SQLite.
  • Embedded in HTML: PHP code is executed on the server, generating HTML to display in the browser.
  • Open Source: Free to use and constantly improved by a global community of developers.
  • Loosely Typed: PHP doesnâ€™t require variable type declarations, making it easier to write flexible code.

Common Use Cases

  • Dynamic Website Development: Creating pages that display different data each time they are loaded.
  • Content Management Systems (CMS): Popular systems like WordPress, Drupal, and Joomla are written in PHP.
  • Form Handling: Processing user input from forms, such as login information or data submission.
  • E-commerce Platforms: Used in back-end development for online stores, like WooCommerce.
  • Web Applications: Supporting a wide range of web apps that rely on real-time data or interactions with databases.

Popular Frameworks

  • Laravel: A modern PHP framework known for its elegant syntax and ease of use.
  • Symfony: A robust framework ideal for large-scale web applications.
  • CodeIgniter: Lightweight and fast for simple, scalable applications.
  • Zend Framework: A robust, secure, and scalable PHP framework for enterprise applications.
  • CakePHP: Provides a quick and flexible way to build web applications with minimal setup.

PHP Versions

  • PHP 4 (2000): Introduced object-oriented programming (OOP) features.
  • PHP 5 (2004): Expanded OOP support, added exception handling, and enhanced performance.
  • PHP 7 (2015): Drastically improved performance, memory usage, and introduced scalar type hints.
  • PHP 8 (2020): Added JIT compilation for better performance and new language features like union types and named arguments.

Key PHP Functions

  • echo(): Outputs strings, numbers, and HTML.
  • isset(): Checks if a variable is set and not null.
  • empty(): Determines if a variable is empty.
  • array(): Creates an array in PHP.
  • include() and require(): Includes the content of one PHP file into another.

PHP in the Real World

PHP powers some of the most popular websites in the world, including Facebook, WordPress, and Wikipedia. It is especially favored for its ease of use, flexibility, and its ability to integrate with databases, making it a perfect choice for web development. The language continues to evolve, adapting to modern web standards, and remains one of the most used server-side languages in the world.

Cheat Sheet

Basic Level

1. Variables:

In PHP, variables are declared with a $ symbol.

                            $name = "Sahil";
                            $age = 23;

2. Echo Statement:

Used to output strings, numbers, or HTML.

                            echo "Hello, World!";

3. Concatenation:

Combine strings using the dot . operator.

                            echo "Hello, " . $name;

4. If-Else Statements:

Basic conditional logic.

                            if ($age > 18) {
                                echo "Adult";
                            } else {
                                echo "Minor";
                            }

5. Arrays:

Create indexed arrays.

                            $fruits = array("Apple", "Banana", "Orange");
                            echo $fruits[0]; // Outputs "Apple"

6. Associative Arrays:

Arrays with key-value pairs.

                            $person = array("name" => "Sahil", "age" => 23);
                            echo $person['name']; // Outputs "Sahil"

7. Loops:

Basic for loop.

                            for ($i = 0; $i < 5; $i++) {
                                echo $i;
                            }

8. Functions:

Declare simple functions.

                            function greet($name) {
                                echo "Hello, " . $name;
                            }
                            greet("Sahil");

9. Include & Require:

Include external files.

                            include 'header.php';
                            require 'config.php';

10. Form Handling:

Get form data with $_POST.

                            $name = $_POST['name'];
                            echo "Welcome, " . $name;

Intermediate Level

1. Switch Statement:

Alternative to multiple if-else blocks.

                            $day = "Monday";
                            switch ($day) {
                                case "Monday":
                                    echo "Start of the week";
                                    break;
                                case "Friday":
                                    echo "End of the week";
                                    break;
                                default:
                                    echo "Midweek";
                            }

2. Associative Array Loops:

Loop through associative arrays.

                            foreach ($person as $key => $value) {
                                echo $key . ": " . $value;
                            }

3. Multidimensional Arrays:

Arrays within arrays.

                            $students = array(
                                array("name" => "Sahil", "age" => 23),
                                array("name" => "John", "age" => 24)
                            );
                            echo $students[0]['name']; // Outputs "Sahil"

4. String Functions:

Common string operations.

                            $str = "Hello World";
                            echo strlen($str); // Outputs 11
                            echo str_replace("World", "PHP", $str); // Replaces "World" with "PHP"

5. Date Functions:

Get and format the current date.

                            echo date("Y-m-d"); // Outputs current date in "YYYY-MM-DD" format

6. File Handling:

Read and write to files.

                            $file = fopen("data.txt", "r");
                            echo fread($file, filesize("data.txt"));
                            fclose($file);

7. Cookies:

Set and retrieve cookies.

                            setcookie("user", "Sahil", time() + (86400 * 30)); // 30 days
                            echo $_COOKIE['user']; // Outputs "Sahil"

8. Sessions:

Start and use sessions.

                            session_start();
                            $_SESSION['username'] = "Sahil";
                            echo $_SESSION['username']; // Outputs "Sahil"

9. Error Handling:

Use try-catch for exceptions.

                            try {
                                // Some code
                                throw new Exception("Error occurred");
                            } catch (Exception $e) {
                                echo $e->getMessage();
                            }

10. Filters:

Validate input data.

                            $email = "test@example.com";
                            if (filter_var($email, FILTER_VALIDATE_EMAIL)) {
                                echo "Valid email";
                            } else {
                                echo "Invalid email";
                            }

Expert Level

1. Object-Oriented Programming:

Create classes and objects.

                            class Car {
                                public $brand;
                                public function __construct($brand) {
                                    $this->brand = $brand;
                                }
                                public function getBrand() {
                                    return $this->brand;
                                }
                            }
                            $car = new Car("Tesla");
                            echo $car->getBrand(); // Outputs "Tesla"

2. Inheritance:

Extend classes.

                            class ElectricCar extends Car {
                                public $batteryRange;
                                public function __construct($brand, $batteryRange) {
                                    parent::__construct($brand);
                                    $this->batteryRange = $batteryRange;
                                }
                            }
                            $myCar = new ElectricCar("Tesla", 300);
                            echo $myCar->getBrand(); // Outputs "Tesla"

3. Namespaces:

Avoid name conflicts.

                            namespace Vehicles;
                            class Car {
                                // Class code
                            }

4. Interfaces:

Define a contract for classes.

                            interface Vehicle {
                                public function drive();
                            }
                            class Bike implements Vehicle {
                                public function drive() {
                                    echo "Bike is driving";
                                }
                            }

5. Abstract Classes:

Blueprint for subclasses.

                            abstract class Animal {
                                abstract public function sound();
                            }
                            class Dog extends Animal {
                                public function sound() {
                                    echo "Bark";
                                }
                            }

6. Traits:

Reuse code across classes.

                            trait Logger {
                                public function log($message) {
                                    echo $message;
                                }
                            }
                            class App {
                                use Logger;
                            }
                            $app = new App();
                            $app->log("Logging in PHP");

7. Regular Expressions:

Match patterns in strings.

                            $pattern = "/php/i";
                            $text = "I love PHP";
                            echo preg_match($pattern, $text); // Outputs 1 (true)

8. PDO for Database:

Secure database interaction.

                            $pdo = new PDO('mysql:host=localhost;dbname=test', 'user', 'password');
                            $stmt = $pdo->prepare("SELECT * FROM users WHERE id = ?");
                            $stmt->execute([1]);
                            $user = $stmt->fetch();

9. Dependency Injection:

Inject dependencies into objects.

                            class Car {
                                protected $engine;
                                public function __construct(Engine $engine) {
                                    $this->engine = $engine;
                                }
                            }

10. Design Patterns:

Implement common design patterns like Singleton.

                            class Singleton {
                                private static $instance;
                                private function __construct() {}
                                public static function getInstance() {
                                    if (self::$instance == null) {
                                        self::$instance = new Singleton();
                                    }
                                    return self::$instance;
                                }
                            }

Singleton Pattern

Implement the singleton pattern in PHP.

            <?php
            class Singleton {
                private static $instance;
                private function __construct() {}
                public static function getInstance() {
                    if (self::$instance === null) {
                        self::$instance = new Singleton();
                    }
                    return self::$instance;
                }
            }
            $singleton = Singleton::getInstance();
            ?>

Recursive Function

Use recursive functions in PHP to solve complex problems.

            <?php
            function factorial($number) {
                if ($number <= 1) {
                    return 1;
                }
                return $number * factorial($number - 1);
            }
            echo factorial(5); // Outputs 120
            ?>

            }
            ?>

            }
            ?>
