CSS Resources

Language Snapshot

Overview

  • Definition: Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML or XML.
  • Purpose: It enables the separation of content and presentation, allowing for more flexible styling and improved accessibility.

Invention

  • Inventor: HÃ¥kon Wium Lie
  • First Released: December 1996
  • Evolution: CSS has evolved through several versions, with CSS3 being the most current and widely used version.

Key Features

  • Selectors: Define which elements to style.
  • Box Model: Describes the rectangular boxes generated for elements, including margins, borders, padding, and content.
  • Responsive Design: Enables designs that adapt to different screen sizes and resolutions.
  • Animations: Provides capabilities for animating changes to CSS properties.

Basic Syntax


            selector {
                property: value;
            }

Common Uses

  • Styling Text: Fonts, colors, and text alignment.
  • Layout Design: Positioning elements, creating grids and flex layouts.
  • Visual Effects: Adding shadows, transitions, and animations.

Cheat Sheet

Basic Level

1. Select all elements:

Sets default styles for all elements.

            * {
                margin: 0;
                padding: 0;
            }

2. Set background color:

Sets the background color of the body element.

            body {
                background-color: lightblue;
            }

3. Text color:

Sets the text color of paragraph elements.

            p {
                color: darkblue;
            }

4. Font size:

Sets the font size of heading 1 elements.

            h1 {
                font-size: 24px;
            }

5. Centering text:

Centers the text of elements with the class "center".

            .center {
                text-align: center;
            }

Intermediate Level

1. Flexbox layout:

Creates a flexbox layout with justified content.

            .container {
                display: flex;
                justify-content: space-between;
            }

2. Grid layout:

Creates a grid layout with three equal columns.

            .grid {
                display: grid;
                grid-template-columns: repeat(3, 1fr);
            }

3. Media query for responsiveness:

Changes the background color of the body element on smaller screens.

            @media (max-width: 600px) {
                body {
                    background-color: lightgreen;
                }
            }

4. Hover effect:

Changes the text color and adds an underline on hover.

            a:hover {
                color: red;
                text-decoration: underline;
            }

5. Transition effect:

Adds a transition effect to the background color of buttons.

            button {
                transition: background-color 0.3s;
            }
            
            button:hover {
                background-color: yellow;
            }

Expert Level

1. CSS Variables:

Defines a CSS variable and uses it to set the background color.

            :root {
                --main-color: coral;
            }
            
            div {
                background-color: var(--main-color);
            }

2. Keyframe animation:

Creates a keyframe animation that changes the background color.

            @keyframes example {
                from {background-color: red;}
                to {background-color: yellow;}
            }
            
            div {
                animation: example 4s infinite;
            }

3. Custom scrollbar:

Customizes the appearance of the scrollbar.

            ::-webkit-scrollbar {
                width: 12px;
            }
            
            ::-webkit-scrollbar-thumb {
                background: darkgrey;
            }

4. Z-index stacking:

Demonstrates z-index stacking with two boxes.

            .box1 {
                position: relative;
                z-index: 1;
            }
            
            .box2 {
                position: relative;
                z-index: 2;
            }

5. CSS Grid with Named Areas:

Creates a grid layout with named areas.

            .grid-container {
                display: grid;
                grid-template-areas: 
                    'header header header'
                    'sidebar content content'
                    'footer footer footer';
            }

Interview Sheet

Standard Code Snippets

Basic Level

Basic Styling

Basic styling for the body element:

                          body {
                              font-family: Arial, sans-serif;
                              background-color: #f4f4f4;
                              color: #333;
                          }
Centering Text

Centering text using the text-align property:

                          h1 {
                              text-align: center;
                          }
Styling a Button

Basic styling for a button:

                          .button {
                              background-color: #007bff;
                              color: white;
                              padding: 10px 20px;
                              border: none;
                              border-radius: 5px;
                              cursor: pointer;
                          }
Hover Effect

Adding a hover effect to a button:

                          .button:hover {
                              background-color: #0056b3;
                          }
Adding Margins

Adding margins to a container:

                          .container {
                              margin: 20px;
                          }

Intermediate Level

Flexbox Layout

Basic flexbox layout:

                          .flex-container {
                              display: flex;
                              justify-content: space-between;
                              align-items: center;
                          }
Grid Layout

Basic grid layout:

                          .grid-container {
                              display: grid;
                              grid-template-columns: repeat(3, 1fr);
                              gap: 10px;
                          }
Media Queries for Responsive Design

Using media queries for responsive design:

                          @media (max-width: 600px) {
                              .flex-container {
                                  flex-direction: column;
                              }
                          }
Custom Scrollbar

Customizing the scrollbar:

                          ::-webkit-scrollbar {
                              width: 10px;
                          }
              
                          ::-webkit-scrollbar-thumb {
                              background: #888;
                              border-radius: 10px;
                          }
Transitions

Adding transitions to an element:

                          .box {
                              width: 100px;
                              height: 100px;
                              background-color: blue;
                              transition: background-color 0.3s;
                          }
              
                          .box:hover {
                              background-color: red;
                          }

Expert Level

CSS Animation

Creating a CSS animation:

                          @keyframes slide {
                              from {
                                  transform: translateX(0);
                              }
                              to {
                                  transform: translateX(100%);
                              }
                          }
              
                          .animate {
                              animation: slide 1s forwards;
                          }
Creating a Responsive Navbar

Creating a responsive navbar:

                          nav {
                              display: flex;
                              justify-content: space-between;
                              align-items: center;
                              background-color: #333;
                          }
              
                          nav ul {
                              display: flex;
                              list-style: none;
                              padding: 0;
                          }
              
                          nav ul li {
                              margin: 0 15px;
                          }
              
                          nav ul li a {
                              color: white;
                              text-decoration: none;
                          }
Sticky Footer

Creating a sticky footer:

                          body {
                              display: flex;
                              flex-direction: column;
                              min-height: 100vh;
                          }
              
                          footer {
                              margin-top: auto;
                          }
Creating a Tooltip

Creating a tooltip:

                          .tooltip {
                              position: relative;
                              display: inline-block;
                          }
              
                          .tooltip .tooltiptext {
                              visibility: hidden;
                              width: 120px;
                              background-color: black;
                              color: white;
                              text-align: center;
                              border-radius: 6px;
                              padding: 5px 0;
                              position: absolute;
                              z-index: 1;
                              bottom: 125%; /* Position the tooltip above the text */
                              left: 50%;
                              margin-left: -60px;
                          }
              
                          .tooltip:hover .tooltiptext {
                              visibility: visible;
                          }
Box Shadows and Text Shadows

Adding box shadows and text shadows:

                          .box-shadow {
                              box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
                          }
              
                          .text-shadow {
                              text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);
                          }
