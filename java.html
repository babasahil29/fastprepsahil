Java Resources

Language Snapshot

Overview

  • Language Type: High-level, class-based, object-oriented
  • Key Features:
    • Object-Oriented: Emphasizes encapsulation, inheritance, polymorphism, and abstraction
    • Platform Independence: "Write once, run anywhere" principle allows compiled code to run on all platforms that support Java
    • Robust and Secure: Strong type checking, exception handling, and security features
    • Multithreaded: Supports concurrent programming for efficient performance
    • Garbage Collection: Automatic memory management system that handles object lifecycle

Invention

  • Creator: James Gosling at Sun Microsystems (now owned by Oracle)
  • Initial Release: May 23, 1995
  • Latest Stable Version: Java SE 17 (as of September 2021)
  • Major Influence: Enterprise-scale applications, Android app development, web applications

Notable Frameworks and Libraries

  • Android SDK: For Android app development
  • Spring Framework: For enterprise-level and web applications
  • JavaFX: For desktop application development

Common Use Cases

  • Android App Development: Creating mobile applications for Android devices
  • Enterprise Applications: Building large-scale systems for banking, retail, and other industries
  • Web Applications: Developing backend services using frameworks like Spring
  • Desktop Applications: Creating cross-platform desktop software

Cheat Sheet

Basic Level

1. Variables

Used to store data values.

            int x = 9;
            String name = "Sahil";

2. Data Types

Defines the type of variable.

            int num = 10; // Integer
            double val = 9.8; // Floating point number
            boolean flag = true; // Boolean

3. If-Else Statement

Conditional logic execution.

            if (x > 0) {
                System.out.println("Positive");
            } else {
                System.out.println("Negative");
            }

4. Loops (For Loop)

Repeats a block of code.

            for (int i = 0; i < 5; i++) {
                System.out.println(i);
            }

5. Methods (Functions)

Blocks of code that perform a task.

            public static void greet() {
                System.out.println("Hello, Java!");
            }

6. Array Declaration

Holds multiple values of the same type.

            int[] numbers = {1, 2, 3, 4, 5};

7. String Manipulation

Strings and basic operations.

            String s = "Java";
            System.out.println(s.length()); // Get length

Intermediate Level

1. Classes and Objects

Java is object-oriented; everything revolves around classes and objects.

            class Car {
                String model = "Honda";
            }
            
            public class Main {
                public static void main(String[] args) {
                    Car myCar = new Car(); // Create an object
                    System.out.println(myCar.model);
                }
            }

2. Constructor

A special method to initialize objects.

            class Car {
                String model;
                Car(String modelName) {
                    model = modelName;
                }
            }

3. Inheritance

Mechanism to acquire properties from a parent class.

            class Vehicle {
                public void run() {
                    System.out.println("Vehicle is running");
                }
            }
            
            class Car extends Vehicle {
                public void run() {
                    System.out.println("Car is running");
                }
            }

4. Polymorphism

One action, many forms.

            Vehicle myCar = new Car();
            myCar.run(); // Outputs "Car is running"

5. Encapsulation

Restrict access using private fields and public methods.

            class Person {
                private String name;
                public String getName() {
                    return name;
                }
                public void setName(String newName) {
                    name = newName;
                }
            }

6. Exception Handling

Managing runtime errors using try-catch.

            try {
                int divide = 10 / 0;
            } catch (ArithmeticException e) {
                System.out.println("Cannot divide by zero");
            }

Expert Level

1. Multithreading

Running multiple threads simultaneously.

            class MyThread extends Thread {
                public void run() {
                    System.out.println("Thread is running");
                }
            }
            
            public class Main {
                public static void main(String[] args) {
                    MyThread thread = new MyThread();
                    thread.start();
                }
            }

2. Lambda Expressions

Used to provide implementation for a functional interface.

            interface Greeting {
                String sayHello(String name);
            }
            
            Greeting greet = (name) -> "Hello, " + name;

3. Streams API

Processing collections of objects.

            List names = Arrays.asList("Java", "Python", "C++");
            names.stream()
                .filter(n -> n.startsWith("J"))
                .forEach(System.out::println);

4. Generics

Write code that can handle different data types.

            class Box {
                private T item;
                public void setItem(T item) {
                    this.item = item;
                }
                public T getItem() {
                    return item;
                }
            }

5. Annotations

Metadata for classes, methods, and variables.

            @Override
            public void run() {
                System.out.println("Running overridden method");
            }

6. Reflection

Inspect or modify runtime behavior of classes, methods, and fields.

            Class cls = Class.forName("java.lang.String");
            Method[] methods = cls.getMethods();

Find the Factorial of a Number

Problem: Calculate the factorial of a given number.

        public class Factorial {
            public static void main(String[] args) {
                int num = 5;
                int result = 1;
                for (int i = 1; i <= num; i++) {
                    result *= i;
                }
                System.out.println("Factorial of " + num + " is: " + result);
            }
        }

Problem: Determine if a given number is prime.

        public class PrimeCheck {
            public static void main(String[] args) {
                int num = 29;
                boolean isPrime = true;
                for (int i = 2; i <= num / 2; i++) {
                    if (num % i == 0) {
                        isPrime = false;

                }
                if (isPrime) {
                    System.out.println(num + " is a prime number.");
                } else {
                    System.out.println(num + " is not a prime number.");

            }

                int n = 10, a = 0, b = 1;
                System.out.print("Fibonacci Series: ");
                for (int i = 1; i <= n; i++) {

Find the Second Largest Element in an Array

Problem: Find the second largest element in a given array.

        public class SecondLargest {
            public static void main(String[] args) {
                int[] arr = {5, 9, 11, 2, 8};
                int first = Integer.MIN_VALUE, second = Integer.MIN_VALUE;
        
                for (int num : arr) {
                    if (num > first) {
                        second = first;
                        first = num;
                    } else if (num > second && num != first) {
                        second = num;
                    }
                }
        
                System.out.println("Second largest element: " + second);
            }
        }
Check for Armstrong Number

Problem: Check if a given number is an Armstrong number.

        public class ArmstrongCheck {
            public static void main(String[] args) {
                int num = 153;
                int originalNum = num, result = 0;
        
                while (originalNum != 0) {
                    int remainder = originalNum % 10;
                    result += Math.pow(remainder, 3);
                    originalNum /= 10;
                }
        
                if (result == num) {
                    System.out.println(num + " is an Armstrong number.");
                } else {
                    System.out.println(num + " is not an Armstrong number.");
                }
            }
        }

Expert Level

Find All Permutations of a String

Problem: Generate all permutations of a given string.

        public class Permutations {
            public static void main(String[] args) {
                String str = "ABC";
                permute(str, 0, str.length() - 1);
            }
        
            private static void permute(String str, int l, int r) {
                if (l == r) {
                    System.out.println(str);
                } else {
                    for (int i = l; i <= r; i++) {
                        str = swap(str, l, i);
                        permute(str, l + 1, r);
                        str = swap(str, l, i); // backtrack
                    }
                }
            }
        
            public static String swap(String a, int i, int j) {
                char[] charArray = a.toCharArray();
                char temp = charArray[i];
                charArray[i] = charArray[j];
                charArray[j] = temp;
                return String.valueOf(charArray);
            }
        }
Implement a Binary Search Algorithm

Problem: Implement binary search algorithm to find an element in a sorted array.

        public class BinarySearch {
            public static int binarySearch(int[] arr, int left, int right, int key) {
                if (right >= left) {
                    int mid = left + (right - left) / 2;
        
                    if (arr[mid] == key)
                        return mid;
        
                    if (arr[mid] > key)
                        return binarySearch(arr, left, mid - 1, key);
        
                    return binarySearch(arr, mid + 1, right, key);
                }
        
                return -1;
            }
        
            public static void main(String[] args) {
                int[] arr = {2, 3, 4, 10, 40};
                int key = 10;
                int result = binarySearch(arr, 0, arr.length - 1, key);
                if (result == -1)
                    System.out.println("Element not present");
                else
                    System.out.println("Element found at index " + result);
            }
        }
Find the Longest Substring Without Repeating Characters

Problem: Find the longest substring without repeating characters in a given string.

        import java.util.HashSet;
        
        public class LongestSubstring {
            public static void main(String[] args) {
                String str = "abcabcbb";
                System.out.println("Longest Substring Without Repeating Characters: " + findLongestSubstring(str));
            }
        
            public static int findLongestSubstring(String str) {
                int n = str.length();
                int result = 0;
        
                for (int i = 0; i < n; i++) {
                    HashSet set = new HashSet<>();
                    for (int j = i; j < n; j++) {
                        if (set.contains(str.charAt(j)))
                            break;
                        set.add(str.charAt(j));
                        result = Math.max(result, j - i + 1);
                    }
                }
                return result;
            }
        }
